# YouTube Recommender ML Project

This project simulates a YouTube-style recommendation system using neural networks.  

## Features
- Baseline recommender
- Neural network model
- Recall@10 metric

## Requirements
- Python 3.14
- PyTorch with CUDA 13 (GPU required)
- numpy, pandas, scikit-learn, matplotlib, tqdm, streamlit

## Usage
1. Clone the repo
2. Activate virtual environment
3. Run `streamlit run app.py` to see recommendations
